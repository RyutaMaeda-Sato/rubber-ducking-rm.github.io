TypeScriptでの文字列検索：用途別チートシート

2025-03-25

これは、TypeScriptでの文字列検索と操作のための簡潔なガイドです。よくある用途別に整理しているため、必要な実装をすぐに見つけることができます。

基本的な文字検索

単一の文字/部分文字列を検索

// 1. includes() メソッド - 最も推奨される方法
function containsCharacter(text: string, char: string): boolean {
  return text.includes(char);
}

// 2. indexOf() メソッド - 古いブラウザ互換性が必要な場合
function hasCharacter(text: string, char: string): boolean {
  return text.indexOf(char) !== -1;
}

// 使用例
const text = "Hello, TypeScript!";
console.log(containsCharacter(text, "Type")); // true

大文字/小文字を区別しない検索

function containsIgnoreCase(text: string, search: string): boolean {
  return text.toLowerCase().includes(search.toLowerCase());
}

// 使用例
console.log(containsIgnoreCase("Hello", "hello")); // true

パターンマッチング

正規表現を使った検索

// 基本パターン検索
function matchesPattern(text: string, pattern: string): boolean {
  const regex = new RegExp(pattern);
  return regex.test(text);
}

// 文字クラスの使用例
const hasDigit = (text: string): boolean => /\d/.test(text);
const hasLetter = (text: string): boolean => /[a-zA-Z]/.test(text);
const hasSpecialChar = (text: string): boolean => /[!@#$%^&*()]/.test(text);

// エスケープが必要な特殊文字の検索
function findSpecialChar(text: string, specialChar: string): boolean {
  const escapedChar = specialChar.replace(/[.*+?^${}()|[\]\\]/g, '\\$&');
  return new RegExp(escapedChar).test(text);
}

複数文字の検索

複数文字のいずれかを検索

function containsAny(text: string, searchTerms: string[]): boolean {
  return searchTerms.some(term => text.includes(term));
}

// 使用例
console.log(containsAny("Hello", ["x", "H", "z"])); // true (Hが含まれている)

すべての文字が含まれているか確認

function containsAll(text: string, searchTerms: string[]): boolean {
  return searchTerms.every(term => text.includes(term));
}

// 使用例
console.log(containsAll("Hello World", ["Hello", "World"])); // true
console.log(containsAll("Hello World", ["Hello", "TypeScript"])); // false

入力バリデーション

パスワード強度チェック

function validatePassword(password: string): { valid: boolean; messages: string[] } {
  const checks = [
    { test: (p: string) => p.length >= 8, message: "8文字以上必要です" },
    { test: (p: string) => /[A-Z]/.test(p), message: "大文字が必要です" },
    { test: (p: string) => /[a-z]/.test(p), message: "小文字が必要です" },
    { test: (p: string) => /[0-9]/.test(p), message: "数字が必要です" },
    { test: (p: string) => /[!@#$%^&*]/.test(p), message: "特殊文字が必要です" }
  ];

  const failedChecks = checks
    .filter(check => !check.test(password))
    .map(check => check.message);

  return {
    valid: failedChecks.length === 0,
    messages: failedChecks
  };
}

禁止文字チェック

function hasForbiddenChars(text: string, forbiddenChars: string[]): string[] {
  return forbiddenChars.filter(char => text.includes(char));
}

// 使用例：XSS対策
const userInput = "<script>alert('XSS')</script>";
const forbidden = ["<", ">", "script"];
const found = hasForbiddenChars(userInput, forbidden);
console.log(found); // ["<", ">", "script"]

テキスト分析

文字の出現回数をカウント

function countOccurrences(text: string, char: string): number {
  let count = 0;
  let pos = text.indexOf(char);

  while (pos !== -1) {
    count++;
    pos = text.indexOf(char, pos + 1);
  }

  return count;
}

// より簡潔な代替実装
function countOccurrencesAlt(text: string, char: string): number {
  return text.split(char).length - 1;
}

文字の位置を全て取得

function findAllPositions(text: string, char: string): number[] {
  const positions: number[] = [];
  let pos = text.indexOf(char);

  while (pos !== -1) {
    positions.push(pos);
    pos = text.indexOf(char, pos + 1);
  }

  return positions;
}

パフォーマンス最適化

早期リターン

function efficientSearch(text: string, terms: string[]): boolean {
  for (const term of terms) {
    if (text.includes(term)) {
      return true; // 一致が見つかった時点で終了
    }
  }
  return false;
}

検索結果のキャッシング

// 同じ検索を繰り返し行う場合
const searchCache = new Map<string, boolean>();

function cachedSearch(text: string, search: string): boolean {
  const cacheKey = `${text}:${search}`;

  if (searchCache.has(cacheKey)) {
    return searchCache.get(cacheKey)!;
  }

  const result = text.includes(search);
  searchCache.set(cacheKey, result);
  return result;
}

大きなテキストの処理

function searchLargeText(text: string, search: string): boolean {
  const chunkSize = 10000;

  for (let i = 0; i < text.length; i += chunkSize) {
    // searchがチャンクをまたがる可能性を考慮し、オーバーラップさせる
    const chunk = text.slice(i, i + chunkSize + search.length - 1);
    if (chunk.includes(search)) {
      return true;
    }
  }

  return false;
}

特殊ケースの処理

Unicode文字とエモジの処理

// 正しいUnicode文字数をカウント
function countUnicodeChars(text: string): number {
  return [...text].length;
}

// エモジを含む文字列かチェック
function containsEmoji(text: string): boolean {
  // 一般的なエモジのUnicode範囲
  const emojiRegex = /[\u{1F300}-\u{1F6FF}\u{1F900}-\u{1F9FF}]/u;
  return emojiRegex.test(text);
}

文字列置換

// 全ての出現を置換
function replaceAll(text: string, search: string, replace: string): string {
  return text.split(search).join(replace);
}

// ES2021以降はreplaceAllネイティブメソッドが使える
function modernReplaceAll(text: string, search: string, replace: string): string {
  return text.replaceAll(search, replace);
}

実践的なツール関数

ユーザー入力のサニタイズ

function sanitizeInput(input: string): string {
  // HTML特殊文字をエスケープ
  return input
    .replace(/&/g, "&amp;")
    .replace(/</g, "&lt;")
    .replace(/>/g, "&gt;")
    .replace(/"/g, "&quot;")
    .replace(/'/g, "&#039;");
}

URLからクエリパラメータを抽出

function getQueryParam(url: string, param: string): string | null {
  const searchParams = new URL(url).searchParams;
  return searchParams.get(param);
}

// 使用例
console.log(getQueryParam("https://example.com/?id=123&name=test", "id")); // "123"
